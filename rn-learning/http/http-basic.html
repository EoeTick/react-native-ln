WHAT IS HTTP

HTTP: HyperText Transfer Protocol 超文本传输协议

HTTP是一个客户端和服务端请求和应答的标准（TCP）。

HTTP是基于TCP/Ip协议的应用层协议。它不涉及数据包（packet）传输，主要规定了客户端和服务端之间的通信格式。

客户端是终端用户，服务器端是网站。通过使用Web浏览器等，客户端发起一个到服务器上指定端口（默认端口为80）的HTTP请求。

URL and URI

URL: uniform resource location 统一资源定位符

URI: uniform resource identifier 统一资源标识符

URI是一种资源的标识；而URL也是一种URI。

URI是一种抽象的资源标识，既可以绝对的，也可以是相对的。但URL指明了定位的信息，必须是绝对的。(缺点)
